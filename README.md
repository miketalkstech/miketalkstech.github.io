# MikeTalksTech

## Setup
1. Upload all files and folders to your `miketalkstech.github.io` repo
2. Add `logo.png` and `photo.jpg` to the `images/` folder
3. Replace `YOUR_FORM_ID` in `contact/index.html` with your Formspree form ID
4. Optionally set custom domain in Settings > Pages

## File Structure
```
index.html          Home (miketalkstech.com/)
style.css           All styles
sitemap.xml         SEO sitemap
images/
  logo.png
  photo.jpg
about/
  index.html        About (miketalkstech.com/about/)
classes/
  index.html        Classes (miketalkstech.com/classes/)
contact/
  index.html        Contact (miketalkstech.com/contact/)
```

## Clean URLs
Pages use folder structure so URLs are clean:
- miketalkstech.com/about/ (not about.html)
- miketalkstech.com/classes/
- miketalkstech.com/contact/

## Design
- Sora (headings) + DM Sans (body)
- #0C314B dark blue + #3B82F6 electric blue
- 800px max width
- Plain HTML/CSS, no dependencies
